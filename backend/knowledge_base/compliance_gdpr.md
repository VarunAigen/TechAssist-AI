# CloudNexus — Compliance & Data Privacy

## Compliance Certifications

### SOC 2 Type II
CloudNexus has achieved SOC 2 Type II certification, audited annually by an independent third-party auditor. The certification covers:
- Security
- Availability
- Processing Integrity
- Confidentiality
- Privacy

SOC 2 reports are available to Enterprise customers upon request under NDA.

### ISO 27001
CloudNexus is ISO 27001 certified, demonstrating our commitment to information security management best practices.

### PCI DSS Level 1
For customers processing payment data, CloudNexus infrastructure is PCI DSS Level 1 compliant. This is the highest level of certification in the payment card industry.

### HIPAA
CloudNexus offers HIPAA-compliant infrastructure for healthcare customers. Enterprise customers can sign a Business Associate Agreement (BAA) to use CloudNexus for Protected Health Information (PHI).

**HIPAA features include:**
- Encrypted data storage (AES-256)
- Access audit logging
- Automatic session timeouts
- Role-based access controls
- Data backup and disaster recovery

## GDPR Compliance

CloudNexus is fully compliant with the EU General Data Protection Regulation (GDPR).

### Data Processing
- CloudNexus acts as a Data Processor on behalf of customers (Data Controllers)
- A Data Processing Agreement (DPA) is available for all customers
- Sub-processors are listed at https://cloudnexus.io/legal/sub-processors

### Data Subject Rights
CloudNexus provides tools for customers to fulfill data subject requests:
- **Right to Access**: Export all user data via API or dashboard
- **Right to Deletion**: Delete user data with the data erasure tool
- **Right to Portability**: Export data in standard formats (JSON, CSV)
- **Right to Rectification**: Update user records via API

### Data Residency
Enterprise customers can restrict data processing to specific regions:
- EU-only processing (Frankfurt, Ireland, London)
- US-only processing (Virginia, Oregon)
- APAC-only processing (Singapore, Tokyo, Sydney)

### Data Retention
- Active account data: Retained while account is active
- Deleted account data: Purged within 30 days of account deletion
- Backup data: Purged within 90 days of deletion
- Audit logs: Retained for 1 year (non-configurable for compliance)

## Privacy Policy
Our full privacy policy is available at https://cloudnexus.io/privacy. Key points:
- We never sell customer data
- We do not use customer data for advertising
- Customer data is only accessed by authorized personnel for support purposes
- All access to customer data is logged and audited
